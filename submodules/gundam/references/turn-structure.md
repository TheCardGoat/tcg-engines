# Turn Structure & Setup

Reference for game setup, turn phases, and phase-specific rules.  
Source: Gundam Card Game Comprehensive Rules Ver. 1.5.0

---

## Setup (Section 6-2)

1. Shuffle both the main deck (50 cards) and resource deck (10 cards).
2. Each player draws 5 cards.
3. Optional redraw (mulligan).
4. Place 6 shields face-down in the shield area.
5. Place the EX Base.
6. Player Two receives an EX Resource.

---

## Turn Phases (Section 7)

Each turn proceeds through five phases in order:

### 1. Start Phase

- Beginning-of-turn triggers resolve.
- Ready all rested cards (unless an effect prevents it).

### 2. Draw Phase

- Active player draws 1 card from their deck.

### 3. Resource Phase

- Active player may place 1 resource from the resource deck into the resource area.

### 4. Main Phase

The active player may perform any number of the following actions in any order:

- **Play cards** — Pay a card's cost by resting resources equal to the printed cost. The player must have enough active resources at the required level.
- **Activate effects** — Use 【Activate·Main】 or 【Main】 abilities.
- **Declare attacks** — See [combat-and-battles.md](combat-and-battles.md).

### 5. End Phase

- End-of-turn effects resolve (e.g., `<Repair>` healing).
- Discard down to hand limit (10 cards) if over.
- Turn passes to the opponent.

---

## Key Phase Rules

- Units **cannot attack on the turn they are deployed** unless they enter play via a link (pairing a pilot that satisfies the link condition) or an effect explicitly grants them the ability.
- A player whose deck reaches 0 cards is defeated during rules management — this can occur mid-phase.
- The active player always resolves simultaneous choices first (Rule 1-3).
