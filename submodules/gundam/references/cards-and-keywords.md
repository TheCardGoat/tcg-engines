# Cards & Keywords

Reference for card types, keyword effects, timing keywords, pilots, and linking.  
Source: Gundam Card Game Comprehensive Rules Ver. 1.5.0

---

## Card Types (Section 3)

### Unit (3-2)

- Deployed to the **battle area** (max 6 Units).
- The only card type that can **attack**.
- Has **AP** (Attack Points) and **HP** (Hit Points).
- Cannot attack the turn it enters play unless it enters via a **link** or an effect overrides this.
- A Unit at 0 HP is **destroyed**.

### Pilot (3-3)

- Paired **under** a Unit in the battle area.
- Adds stat bonuses (AP/HP) and effects to the linked Unit.
- Moves with the Unit whenever the Unit changes zones.

### Command (3-4)

- One-time effect card.
- Sent to the **trash** after resolution.
- May have 【Pilot】 (restricted to a named pilot's Unit) or 【Burst】 (activates when flipped from shields).

### Base (3-5)

- Placed in the **shield area**.
- Absorbs damage before regular shields.
- Has HP; destroyed when HP reaches 0.

### Resource (3-6)

- Placed in the **resource area** from the resource deck.
- Rested to pay card costs.
- Has no color.

---

## Linking & Pairing

- **Pair**: Place a Pilot card under a Unit. The Pilot's AP/HP bonuses apply.
- **Link**: When a Unit's **link condition** is met by the paired Pilot, the Unit is considered "linked," gaining additional effects and the ability to attack the turn it enters play.
- **Link Condition** (2-12): Printed requirements specifying which Pilot(s) can link with the Unit (e.g., trait, name, color).

## Keyword Effects (Section 13)

| Keyword           | Effect                                                    |
| ----------------- | --------------------------------------------------------- |
| `<Repair>`        | Heal damage at end of turn                                |
| `<Breach N>`      | After destroying an enemy Unit, deal N damage to shields  |
| `<Support>`       | Buff allied Units                                         |
| `<Blocker>`       | Rest this Unit to redirect an opponent's attack to itself |
| `<First Strike>`  | Deal damage before the opponent in battle                 |
| `<High-Maneuver>` | Cannot be blocked                                         |
| `<Suppression N>` | Hit N shields instead of 1 on a direct attack             |

## Timing Keywords (Section 13)

| Keyword             | When it fires                                  |
| ------------------- | ---------------------------------------------- |
| 【Activate·Main】   | Activated ability: pay cost during Main Phase  |
| 【Activate·Action】 | Activated ability: pay cost during Action Step |
| 【Main】            | Command card played during Main Phase          |
| 【Action】          | Command card played during Action Step         |
| 【Burst】           | When this card's shield is flipped by damage   |
| 【Deploy】          | When this card enters the battle area          |
| 【Attack】          | When this Unit declares an attack              |
| 【Destroyed】       | When this card is destroyed and sent to trash  |
| 【When Paired】     | At the moment a Pilot is paired with a Unit    |
| 【During Pair】     | Continuous while a Unit has a paired Pilot     |
| 【When Linked】     | At the moment a Unit links with a Pilot        |
| 【During Link】     | Continuous while a Unit is linked              |
| 【Once per Turn】   | This effect can only activate once per turn    |

## Card Properties

- **Card Number** (2-1): Unique identifier. Max 4 copies per deck.
- **Color** (2-4): Blue, Green, Red, White, Purple. Resources and tokens have no color.
- **Trait** (2-5): Categorical tags (e.g., group, class). Cards may have multiple.
- **Level** (2-9): Resource-level requirement to play the card.
- **Cost** (2-10): Number of resources to rest to pay for the card.
- **Card text overrides base rules** when they conflict (Rule 1-3).
