# Lorcana Sets 2-3 Card-by-Card TODO List

> Complete card implementation list for Sets 2-3. This file serves as the master tracking document for implementing all 407 cards from Rise of the Floodborn (Set 2) and Into the Inklands (Set 3).

## Implementation Order

**Phase 1: ALL Actions (Sets 2-3)** - 51 cards
**Phase 2: ALL Items (Sets 2-3)** - 36 cards
**Phase 3: ALL Locations (Set 3 only)** - 18 cards
**Phase 4: ALL Characters (Sets 2-3)** - 302 cards

This order builds pattern libraries efficiently: Actions teach trigger/effect patterns, Items teach durability/static effects, Locations introduce the new card type, and Characters apply all patterns + keywords.

## Implementation Notes

### Card Numbering

- Set 2 cards are numbered 1-204
- Set 3 cards are numbered 1-203
- When implementing, use the collector number from canonical data

### File Naming Convention

```
{cardNumber}-{name-in-kebab-case}.ts
```

Examples:
- `028-hold-still.ts`
- `001-bashful-hopeless-romantic.ts`

### New Card Type: Locations (Set 3 Only)

Set 3 introduces **Location** cards, a new card type not present in Sets 1-2. Key differences:

- Properties: `moveCost`, `willpower`, `lore`
- No `cost` or `inkable` properties
- Special implementation considerations for movement mechanics

### Similarity-Based Implementation

Before implementing any card, use the `pattern-matcher` agent to find similar implemented cards from Set 1 or previously implemented cards. This dramatically speeds up implementation by reusing proven patterns.
